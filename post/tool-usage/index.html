homebrew

替换源

使用homebrew安装软件时，默认会先进行更新，由于国内访问访问github不稳定，导致更新的时候特别慢。可行的方法是替换源，对于homebrew而言，需要替换4个模块的镜像：

  • Homebrew
  • Homebrew Core
  • Homebrew Cask
  • Homebrew-bottles

最全的是中科大镜像，其他家都缺少第4个，所以换了前三个后，执行brew update依然很慢，具体替换方式如下：

# 替换 Homebrew
git -C "$(brew --repo)" remote set-url origin https://mirrors.ustc.edu.cn/brew.git

# 替换 Homebrew Core
git -C "$(brew --repo homebrew/core)" remote set-url origin https://mirrors.ustc.edu.cn/homebrew-core.git

# 替换 Homebrew Cask
git -C "$(brew --repo homebrew/cask)" remote set-url origin https://mirrors.ustc.edu.cn/homebrew-cask.git

# 替换 Homebrew-bottles
# 对于 bash 用户：
echo 'export HOMEBREW_BOTTLE_DOMAIN=https://mirrors.ustc.edu.cn/homebrew-bottles' >> ~/.bash_profile
source ~/.bash_profile
# 对于 zsh 用户：
echo 'export HOMEBREW_BOTTLE_DOMAIN=https://mirrors.ustc.edu.cn/homebrew-bottles' >> ~/.zshrc
source ~/.zshrc

ideviceinstaller

安装App

使用ideviceinstaller安装App，若Mac上只连了一台手机，直接使用如下命令：

ideviceinstaller -i  xxx.ipa

如果连接了多部手机，需要更加设备的UDID来区分，则使用如下命令：

ideviceinstaller -u xxx -i  xxx.ipa

如果要查看连接到电脑上的所有手机的UDID，需要使用如下命令：

idevice_id -l

如果要查看详信息，需要用到ideviceinfo命令，下面的-s选项表示只是展示一些简单的信息：

ideviceinfo -s

查看安装的App

如果要查看手机安装的第三方App，则使用如下命令：

ideviceinstaller -l

卸载应用

ideviceinstaller -u [udid] -U [bundleId]
