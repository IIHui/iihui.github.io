XCode快捷键

创建: 2018-11-27   |   更新: 2018-11-27   |   分类: code     |   字数: 876   |   阅读: 2 分钟

MAC键盘图标

  1. ⌘——Command ()
  2. ⌃ ——Control
  3. ⌥——Option (alt)
  4. ⇧——Shift
  5. ⇪——Caps Lock

快捷键

  1. Command+R 运行
  2. Command+B 编译
  3. Command+. 停止
  4. Command + Shift + K 清除工程
  5. Command + K 清除控制台信息
  6. Command + N 新建文件
  7. Command + Shift + N 创建新工程

导航栏

  1. Command+0~9 工程导航
    1. Command + 0 快速打开实用面板
    2. Command + 1 查看工程文件
    3. Command + 2 版本控制导航
    4. Command + 3 符号导航(类)
    5. Command + 4 警告和错误导航
    6. Command + 5 测试导航
    7. Command + 6 调试导航
    8. Command + 7 断点导航
    9. Command + 9 报告导航
  2. Command + Option + 0 打开右侧导航栏
  3. Command + Option + T 打开工具栏
  4. Command + Shift + Y 打开底部导航栏
  5. Command + Z 撤销
  6. Command + Shift + Z 反撤销

查找与跳转

  1. Command + F 在当前文档中查找
  2. Command + Shift + G 搜索上一处
  3. Command + G 搜索下一处
  4. Command + L 跳转到特定行
  5. Command + Shift + O（字母） 快速查找打开类
  6. Command + Shift + J 快速定位到当前类在项目中的位置
  7. Command + Shift + 0(数字zero)查看帮助文档
  8. Command + Control+↑/Command+Control+↓ 类文件和实现文件之间跳转
  9. Command+Control+←/Command+Control+→ 回到上一个/下一个打开的页面
  10. Command+Option+←/Command+Option+→ 收放方法体，即将方法体收起或者展开
  11. Control + 6 在当前类查找变量和方法 「键入方法或者变量名+Enter跳转」

光标操作

  1. Control + F 向右一个字符
  2. Control + B 向左一个字符
  3. Control + P 向上一行
  4. Control + N 向下一行
  5. Control + A 跳到行首
  6. Control + E 跳到行尾
  7. Control + T 调换光标两边的字符
  8. Control + D 删除光标右侧的单个字符
  9. Control + K 删除本行剩余的字符

整体位移

  1. Command + [ 选中的代码整体左移
  2. Command + ] 选中的代码整体右移
  3. Command + Option + [ 选中的代码整体上移
  4. Command + Option + ] 选中的代码整体下移

补充

  1. Command + Option + 点击桌面上空白的区域:把除Finder窗口以外的所有窗口隐藏起来，直接显示桌面。分别点击Dock栏上各个相关应用程序图标就可以解除窗口的隐藏状态了。

  2. Command + Option + H: 把所有已打开的程序窗口（不包括当前正在运行的应用程序窗口）最小化到Dock栏上。注意不是关闭哦，是最小化。如果需要把程序窗口恢复到屏幕上，直接点击相应的应用程序图标就可以了。

编译选项

在shema中添加DYLD_PRINT_STATISTICS可统计启动时间
