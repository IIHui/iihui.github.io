CrossOver安装Rider2021.3.3

首先需要安装64位的Win7 Bottle或者64位的Win 10 Bottle，再在新安装的Bottle中安装.NET Framework 4.7.2，这样才能正常安装Rider2021.3.3（依赖关系）。

如果在安装软件后提示未安装宋体，则需要安装简体中文字体包。此时搜索Font，然后选择Adobe Source Han Sans Simplified Chinese Font Package安装即可：

IISExpress

如果想把Rider中的工程发布到IIS Express，可以参考这篇文章。如果下载安装IISExpress后出现Cannot open assembly 'C:/Program Files (x86)/IIS Express/iisexpress.exe': File does not contain a valid CIL image错误。

如果发布的时候出现：IIS express请求已结束: http://localhost:52915/，HTTP状态为403.14，那么就要按照这篇文章操作。

Rider离线安装数据库驱动

因为在内网无法安装数据库驱动，因此需要先从互联网下载驱动jar文件，然后在Data Sources and Drivers中选中某个数据库，再在Driver Files下面选中Custom JARs，导入下载的驱动文件即可：
