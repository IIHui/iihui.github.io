鼠须管设置

时间： 2019-01-08   |   分类： QA     |   阅读： 292 字 ~1分钟

这篇文章是使用鼠须管的设置问题

中英文切换提示

在squirrel.yaml中加入:

good_old_caps_lock: false

候选词横排显示

在squirrel.yaml中加入：

style:
  horizontal: true

候选词字体大小

在squirrel.yaml中加入：

style:
  font_point: 19

候选词数量

在default.custom.yaml中加入：

patch:
  menu:
    page_size: 9  //每页候选词的数量

预定义输入方案

在default.custom.yaml中加入：

schema_list:           #“输入选单”中激活的输入方案定义。
   - schema: luna_pinyin_simp
   - schema: luna_pinyin
   - schema: luna_pinyin_fluency

候选词翻页

在default.custom.yaml中加入：

key_binder:
  bindings:
    - { when: always, accept: Shift+space, toggle: full_shape } # Shift+space 切换全角/半角
    - { when: has_menu, accept: bracketleft, send: Page_Up }     //使用[向上翻页
    - { when: has_menu, accept: bracketright, send: Page_Down }  //使用]向下翻页
