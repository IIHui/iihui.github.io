常量

Kotlin中使用val来声明常量「val是value的缩写」，val声明的常量不可以再重新赋值，否则会报错。

val str = "Hello world"

变量

Kotlin中使用var来声明变量「var是variable的缩写」，var声明的变量可以重新赋值。

var test = "ch"
println(test)
test = "ccc"
println(test)

条件语句

Kotlin不包含三元运算符，更加倾向于使用条件表达式，随着if-else的复杂性增加，可以使用when表达式。

when

Kotlin中的when表达式中每个分支由一个条件，一个箭头和一个结果表示，如果箭头左侧的条件求值为true，则会返回右侧表达式的值。

val count = 43
var str1 = when (count) {
    42 -> "chen"
    56 -> "cchen"
    else -> "test"
}
