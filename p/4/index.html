UIWebview 加载含特殊字符网页

创建: 2019-02-24   |   更新: 2019-02-24   |   分类: code     |   字数: 114   |   阅读: 1 分钟
UIWebview中加载的html文件中如果含有\n,'等特殊字符时，显示为一片空白。查阅资料得知需要将这些特殊字符进行预处理。 预处理 class HTMLUtils{ static func escapsedHTML(html: String)->String{ var result = html if !result.isEmpty{ result = result.replacingOccurrences(of: "\\", with: "\\\\") result = result.replacingOccurrences(of: "\"", with: "\\\"") result = result.replacingOccurrences(of: "'", with: "\\'") result = result.replacingOccurrences(of: "\r", with: "\\r") result = result.replacingOccurrences(of: "\n", with: "\\n") } return result } }
